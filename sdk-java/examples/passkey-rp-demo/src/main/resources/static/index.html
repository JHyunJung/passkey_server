Passkey RP Demo

이 페이지는 passkey-rp-spring-boot-starter SDK를 쓰는 RP 서버의 동작 예시입니다. ceremony 엔드포인트(/passkey/**)는 SDK가 자동 제공합니다.

1. 회원가입

RP가 직접 구현한 POST /users — externalUserId를 발급합니다.

—

2. 패스키 등록

SDK 제공 /passkey/register/begin · /finish

먼저 회원가입을 완료하세요.

3. 패스키 로그인

SDK 제공 /passkey/authenticate/begin · /finish

—

4. 로그인 후 보호된 API

accessToken을 Authorization: Bearer로 전달. /me는 RP 구현, /credentials는 SDK PasskeyClient 위임. 

먼저 로그인하세요.
